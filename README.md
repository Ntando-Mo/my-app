markdown_content = """# Little Lemon Restaurant - Web App

## Overview
This is the capstone project for the Meta Front-End Developer Professional Certificate. It is a responsive, React-based Single Page Application (SPA) for **Little Lemon**, a family-owned Mediterranean restaurant. The core feature is a robust, interactive, and fully validated table reservation system.

## Key Features
- **Multi-Step Booking Wizard:** A user-friendly, two-step reservation form with a dedicated success confirmation screen.
- **Form Validation:** Comprehensive client-side regex and state-based validation to ensure reliable data entry (email, phone, name length).
- **Responsive Design:** Fluid, modern layouts built entirely with CSS Grid and Flexbox, optimized for both desktop and mobile viewports.
- **Client-Side Routing:** Seamless page transitions using `react-router-dom` to prevent browser refreshing.
- **Semantic HTML & Accessibility:** Built with ARIA attributes, semantic structure (`<header>`, `<main>`, `<footer>`), and color-contrast considerations.
- **Custom UI Components:** Premium hover states, active states, and error handling directly built into standard CSS without heavy external libraries.

## Technologies Used
- React.js
- React Router v6
- HTML5 & Modern CSS3
- JavaScript (ES6+)
